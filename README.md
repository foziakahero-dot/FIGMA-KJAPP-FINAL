# Kjapp 🌌

En moderne taxi-app for det norske markedet med AI-assistent Aurora. Holografisk/cinematisk design inspirert av nordlys.

## 🎨 Design

- **Kunde-app**: Cyan/violet aurora-tema
- **Sjåfør-app**: Amber/graphite-tema
- **3 temaer**: Polar Midnight (mørk), Aurora Frost (lys), Obsidian Gold (premium)
- Automatisk tema-bytte basert på soloppgang/solnedgang i Oslo

## ✨ Kule Features

1. **Proaktive Aurora-notifikasjoner** - AI foreslår reiser basert på din rutine
2. **Voice-mode** - Snakk med Aurora
3. **Spotify-overlevering** - Din musikk fortsetter i bilen
4. **Tur-stemning-velger** - Velg atmosfære (stille, prat, fest, power-nap)
5. **Karbon-regnskap** - Spor CO₂-besparelser
6. **Del tur live** - Sanntidsdeling med SOS-funksjon
7. **AR-pickup** - Pek på bilen din med kamera
8. **Aurora-minner** - Spotify Wrapped-stil reisesammendrag
9. **Splitt-betaling** - Vipps-integrasjon for å dele regningen
10. **Sjåfør-rytme** - Pauseplikt og helseovervåking

## 🚀 Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS v4
- **Animasjoner**: Motion (Framer Motion)
- **UI Components**: shadcn/ui
- **Ikoner**: Lucide React
- **Backend** (planlagt): Supabase
- **AI** (planlagt): Claude API

## 🔐 Sjåfør-registrering

Sjåfører kan ikke registrere seg selv - de må inviteres av flåteeiere med invitasjonskode.

## 🛠️ Utvikling

```bash
# Installer avhengigheter
pnpm install

# Start dev-server
pnpm dev

# Build for produksjon
pnpm build
```
